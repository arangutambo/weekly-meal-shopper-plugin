# Sample Assets

Use these files as a publishing/demo starter pack.

## Included

- `sample-weekly-meal-plan.canvas`
- `recipes/Sample Lemon Pasta.md`
- `recipes/Sample Passionfruit Melting Moments.md`

## How to use

1. Copy the sample recipes into your configured recipe folder.
2. Copy `sample-weekly-meal-plan.canvas` into your configured meal-prep canvas folder.
3. Open the canvas and update file links if your vault paths differ.
4. Run `Generate weekly shopping list from meal-plan canvas`.
